yed Install User Guide Developer Manual

Build from source:


Grab the latest source code from www.github.com/kammerdienerb/yed 
                $ git clone git@github.com:kammerdienerb/yed.git
Compile yed using the provided build script. 
                $ ./build.sh
To install yed to the default system install prefix (usually /usr), you can run the install script as-is (you will likely need root privileges). If instead, you'd like to install yed somewhere else, edit install.options to include the desired installation path. 
                $ sudo ./install.sh
NOTE: if you installed yed to a non-default path, ensure that the directory containing the yed executable is in your PATH.
You should now be able to run yed! 
                $ yed # test your installation
