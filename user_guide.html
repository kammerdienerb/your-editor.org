This guide is meant to explain the normal operation and concepts of yed. It assumes that you've already built and installed yed. If you haven't yet, instructions are here.
For information regarding plugin development, see the developer manual.

Starting yed for the first time

yed can be started by simply entering "yed" on the command line. 
                    $ yed # run your editor
